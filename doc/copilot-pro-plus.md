# Copilot Pro+ — Modelos no VS Code

> Guia rápido sobre ativação e acesso aos modelos premium do GitHub Copilot Pro+.

---

## Quanto tempo leva?

Após o pagamento confirmado, o acesso aos modelos premium é **quase imediato**.

| Cenário | Tempo estimado |
|---------|---------------|
| Ativação normal | **Poucos minutos** após a confirmação do pagamento |
| Provisionamento completo | Até **1 hora** em casos raros |
| Se não aparecer em 1h | Verificar os passos de solução abaixo |

---

## Como verificar a assinatura

1. Acesse [github.com/settings/copilot](https://github.com/settings/copilot)
2. Confirme que o plano exibido é **Copilot Pro+**
3. Verifique se o status está **Ativo**

---

## Como acessar os modelos no VS Code

1. **Atualize o VS Code** para a versão mais recente
2. **Atualize a extensão GitHub Copilot** (Extensions → GitHub Copilot → Update)
3. **Faça logout e login** na conta GitHub dentro do VS Code:
   - `Ctrl+Shift+P` → `GitHub Copilot: Sign Out`
   - `Ctrl+Shift+P` → `GitHub Copilot: Sign In`
4. **Selecione o modelo** no painel do Copilot Chat:
   - Abra o Copilot Chat (`Ctrl+Shift+I` ou ícone na barra lateral)
   - No topo do chat, clique no seletor de modelo
   - Escolha entre os modelos premium disponíveis (ex: GPT-4.5, Claude, etc.)

---

## Solução de problemas

Se os modelos não aparecerem após 1 hora:

| Passo | Ação |
|-------|------|
| 1 | Reinicie o VS Code completamente (`Ctrl+Shift+P` → `Developer: Reload Window`) |
| 2 | Verifique se a extensão GitHub Copilot está na versão mais recente |
| 3 | Confira a assinatura em [github.com/settings/copilot](https://github.com/settings/copilot) |
| 4 | Verifique o status do serviço em [githubstatus.com](https://www.githubstatus.com) |
| 5 | Se persistir, reinicie o computador e tente novamente |
| 6 | Último recurso: [Suporte GitHub](https://support.github.com) |

---

## Sobre o Pro+

- **Modelos base** (ex: GPT-4o): acesso ilimitado
- **Modelos premium** (ex: GPT-4.5, Claude Opus): cota de ~1.500 requisições/mês
- Funciona no **Copilot Chat**, **Copilot Edits** e **Agent Mode**
- Disponível no VS Code, Visual Studio, JetBrains e GitHub.com
